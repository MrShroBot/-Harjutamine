Github on koht kus sa saad hostida hoidlaid.
Githubi kasutavad peamiselt programeerida.
Github on koht kus sa saad hostida koodi ja lihtsalt andmeid.

GitHub on Ameerika firma, mis pakub veebimajutusteenust
tarkvaraarenduseks, kasutades Giti tarkvara. GitHub on
Microsofti tütarfirma alates aastast 2018, mil Microsoft
ostis GitHubi 7,5 miljardi dollari eest. GitHub
põhineb Giti hajutatud versioonihaldusel ja lähtekoodihaldusel, 
kuid lisab neile veel võimalusi, näiteks võimalust kontrollida, 
kes pääseb koodile juurde, koodi veakontrolli, tarkvara funktsioonide 
taotlust ning igale projektile oma vikit.
